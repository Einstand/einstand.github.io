FlowSe7en WebAPI
addDynamicApprover
function addDynamicApprover(authToken : string, approverGuid : string, newApproverId : string, newApproverDept : string, options : object);
Description
臨時加簽
Parameters
authToken
權限認證碼
approverGuid
待簽核人唯一碼
newApproverId
新增簽核人的帳號
newApproverDept
新增簽核人的部門代碼
options
選項參數
successfunction執行成功時,所呼叫的函式. 傳入參數如 success(result,status,xhr)
failfunction執行失敗時,另行呼叫的函式. 傳入參數如 fail(result,status,xhr) 或 fail(failMessage)
unauthorizedfunction權限認證碼(authToken)失效. 傳入參數如 unauthorized(result)
errorfunctionajax錯誤時,另行呼叫的函式. 傳入參數如 error(xhr,status,error)

Remarks
無

Examples
var flow7API=window.Flow7API;
var authToken=flow7API.cookie['Flow7Token'];
flow7API.addDynamicApprover(authToken, approverGuid, "Sonia", "QualityMangDept");
